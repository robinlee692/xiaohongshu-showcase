### 3️⃣ 内容生成格式 v3.0

#### 文件结构
```
content/
├── 2026-03-12-ai-tools.md       # Markdown 内容
└── 2026-03-12-ai-tools.prompts.json  # 配图提示词
```

#### Markdown 文件格式
```yaml
---
title: 标题（带 Emoji，15-20 字）
date: 2026-03-12
tags:
  - 标签 1
  - 标签 2
  - 标签 3
status: 待发布
---

正文内容（300-500 字，分 3-4 段）...
```

#### Prompts JSON 文件格式
```json
{
  "imagePrompts": [
    "小红书风格封面，AI 工具推荐主题，明亮渐变色背景（粉紫/橙黄）...",
    "信息图表风格，核心要点分点展示，3-4 个卡片式布局...",
    "真实生活场景，办公桌/咖啡厅环境...",
    "左右/上下对比布局，展示使用前后的差异..."
  ]
}
```

## 📝 使用指南

### 批量删除流程
1. 勾选要删除的内容（或点击"全选"）
2. 点击"删除选中 (N)"按钮
3. 在弹窗中确认删除
4. 删除成功后自动刷新列表

### 切换状态流程
1. 点击卡片/列表中的状态标签
2. 选择新状态（待发布/已发布）
3. 自动保存并同步

### 复制内容流程
1. 进入详情页
2. 找到要复制的部分
3. 点击该部分的"📋 复制"按钮
4. 看到"✅ 已复制"反馈

### 生成新内容流程
```bash
cd /home/robin/.openclaw/workspace/xiaohongshu-showcase
node generate-xiaohongshu-content.js
```

生成后：
1. 检查生成的 `.md` 和 `.prompts.json` 文件
2. Git 提交并推送
3. Vercel 自动部署
