## 📝 使用指南

### 批量删除流程
1. 勾选要删除的内容（或点击"全选"）
2. 点击"删除选中 (N)"按钮
3. 在弹窗中确认删除
4. 删除成功后自动刷新列表

### 切换状态流程
1. 点击卡片/列表中的状态标签
2. 选择新状态（待发布/已发布）
3. 自动保存并同步到文件

### 复制内容流程
1. 进入详情页
2. 找到要复制的部分
3. 点击该部分的"📋 复制"按钮
4. 看到"✅ 已复制"反馈

### 生成新内容流程
```bash
cd /home/robin/.openclaw/workspace/xiaohongshu-showcase
node generate-xiaohongshu-content.js
```
